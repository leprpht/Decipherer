# Decipherer

Projekt zaliczeniowy z przedmiotu Bezpieczeństwo systemów informatycznych

##

Jest aplikacją webową umożliwiającą szyfrowanie i odszyfrowywanie tekstu za pomocą sześciu różnych technik szyfrowania. Program pozwala użytkownikowi zaszyfrowywać teksty za pomocą szyfrów: Cezara, Polibiusza, Vigenère'a, Playfair, Trifid, RSA. Każdy z tych szyfrów jest zaimplementowany jako oddzielna klasa zgodna z interfejsem Cipher, co umożliwia łatwe rozszerzanie aplikacji.

## Tech Stack

- **Frontend**: React with TypeScript and TailwindCSS
- **Build Tool**: Vite
